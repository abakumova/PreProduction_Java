## PreProduction_Java

This project is a result of the completed module of Java during pre-production practice. It consists of 12 tasks (each on a separate branch).

* task1 - Java Basics;
* task2 - OOP;
* task3 - Enum Number String;
* task4 - jUnit, TDD;
* task5 - Log4j;
* task6 - Collections;
* task7 - Multithreading;
* task8 - Java 8;
* task9 - HTTP;
* task10 - Servlet;
* task11 - REST SOAP;
* task12 - JDBC DAO.
